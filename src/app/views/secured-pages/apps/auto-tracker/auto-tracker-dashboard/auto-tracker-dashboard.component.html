Car Cost Dashboard
  • Fuel Activity
This project began as a debate between my father and me about how much it costs to own a car. I decided to track all of my car-related expenses. More recently, I started working on an app that will let me enter these expenses into a MySQL database. I created a dashboard to show interesting charts and statistics about the data being captured.

Technologies: Angular 9, Angular Material, Java/Spring back-end, MySQL database, Chart.js library.
