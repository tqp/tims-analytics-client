Plotting My Planned Running Route Against Position Data Collected
Over the course of one month last year, I wore a GPS tracker while running my standard route around my neighborhood. The goal was to use the data points to create a heatmap that would show when I diverged from that route. Over the course of 13 days, 12691 individual data points were collected (approximately one point every 5 meters). Zooming in, you can see these individual data points. Zooming out, areas in red indicate where I was consistent with my route, where green areas are where I got a bit lost.

Technologies: Leaflet maps, Leaflet.heat plug-in, Angular Typescript code, JSON data.
