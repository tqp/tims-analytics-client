Interactive Globe
This page is an implementation of the Gio.js library, which itself was inspired by the Arms Trade Visualization project developed by Michael Chang and presented during Google Ideas INFO 2012. It pulls data (input country, output country, and volume) from a JSON file. It is also possible to pull and edit this data from an database. Thanks to Altamira for introducing it to me.

Technologies: Angular 9, Gio.js library.
